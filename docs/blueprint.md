# **App Name**: AgroFuturos Conecta

## Core Features:

- Production Management: Enables farmers to record and track all stages of production from planning to post-harvest.
- AI-Powered Projections: Provides farmers with AI-driven tools tool for harvest estimation, cost calculation, and yield prediction.
- Futures Marketplace: Allows farmers to list available and future products for sale, displaying them on a marketplace for buyers.
- Automated Traceability: Generates a 'Product Passport' with a unique QR code for each batch, providing a complete documented history.
- Buyer Offer System: Enables buyers to make offers on future products, specifying price, delivery date, and conditions.
- AI Assistant for Farmers: Offers a chatbot with simple language to answer common farming questions, integrated with LLM models tool.
- Secure Data Storage: Uses Firestore to securely store and manage production, product, offer, and contract data.

## Style Guidelines:

- Primary color: Emerald green (#50C878) evoking growth and agriculture.
- Background color: Pale green (#E0F8E6) for a calm and natural feel.
- Accent color: Sunny yellow (#FFDA63) to highlight important actions.
- Headline font: 'Belleza' (sans-serif) for fashion, art and design and personality
- Body font: 'Alegreya' (serif) for longer texts
- Code font: 'Source Code Pro' for displaying traceability information.
- Use universal icons for key actions (planting, watering, harvesting).
- Design layout with a single call-to-action per screen to provide visual clarity.
- Optimize layout for low-end devices to ensure a smooth experience on less powerful hardware.
- Implement simple animations to confirm successful actions, providing clear feedback to users.
